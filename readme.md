
# Yo la compagnie !

## Instructions pour lancer le projet

1. Installez les dépendances :
   ```bash
   npm install
   ```

2. Lancez le serveur de développement :
   ```bash
   npm run dev
   ```

3. Pour exécuter les tests E2E :
   ```bash
   npm run e2e
   ```

4. Pour exécuter les tests unitaires en mode watch :
   ```bash
   npx vitest --watch
   ```
```
